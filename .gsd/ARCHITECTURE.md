## Data Flow

1. **User navigates** to a page via Next.js app routes.
2. **Page component renders** using static config data and shared UI components.
3. **User actions** (add to cart, theme toggle, navigation) update client state in `useCart`.
4. **Checkout flow** uses `src/app/checkout` and redirects to `checkout/success` on completion.
